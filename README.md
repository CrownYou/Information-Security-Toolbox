这是一款包括了RSA非对称加解密、数字签名、AES对称加解密、哈希计算、隐写术的软件。 软件内的“帮助”部分有详细的使用说明。

激活成功后，软件的登录密码为：1234，登录成功后可以修改登录密码。

注意请不要把exe文件和system_resource文件夹分开，否则软件运行不了哦。您可以为exe文件创建一个快捷方式，然后可以将快捷方式放在任意地方了。

软件只能同时打开一个，否则软件依赖的资源文件会产生冲突。同时打开多个的话，后打开的会自动关闭哦。如果您确实需要打开两个的话，可以将软件和system_resource文件夹复制一份至其他文件夹，再打开。
